# 📡 LoRa Based IoT Communication System using ESP32  

## 📌 Project Overview  
This project demonstrates a long-range wireless IoT communication system using ESP32 and LoRa technology. The system transmits environmental data, voice signals, and image information from a transmitter node to a receiver node.

The receiver displays real-time data on a TFT screen, activates alert mechanisms such as LED and buzzer, and uploads system status to IoT cloud platforms for monitoring and notification.

---

## 🎯 Objectives  
- Design a long-range wireless communication system using LoRa  
- Monitor temperature and humidity remotely  
- Enable voice and image data transmission  
- Generate alert signals during abnormal conditions  
- Upload system status to IoT cloud platforms  

---

## 🔗 Communication Protocols  
- **LoRa** → Long-range wireless communication  
- **SPI** → Interface between ESP32, TFT, and LoRa  
- **UART** → Communication between ESP32 and Raspberry Pi  
- **Wi-Fi / SMTP** → Cloud update and email alert system  

---

## ⚙️ Working Principle  
1. Sensors and modules collect environmental, voice, and image data at the transmitter node.  
2. ESP32 processes the data and sends it via LoRa communication.  
3. The receiver node receives the data and displays it on the TFT screen.  
4. In abnormal conditions, LED and buzzer alerts are activated.  
5. Raspberry Pi uploads data to ThingSpeak IoT cloud.  
6. Gmail alerts are sent to the user for critical conditions.
